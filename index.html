Technical Cloud Projects

Project 1: AWS Serverless Infrastructure Cost Optimizer

The Problem: Companies waste thousands of dollars leaving development and testing servers running 24/7 over weekends and nights when no one is working.

The Solution: An event-driven, automated system that shuts down non-essential development servers at night and turns them back on in the morning.

The Tech Stack: Amazon EventBridge triggers an AWS Lambda function running Python (Boto3) twice a day. The function uses strict AWS IAM permissions to safely identify and toggle Amazon EC2 instances based on specific resource tags (Environment: Development), completely bypassing live production servers.

The Impact: Cuts idle cloud infrastructure compute costs by up to 65% while maintaining a $0 operating footprint by running entirely within the AWS Free Tier.

Project 2: Serverless Student Enrollment Backend (API & Database)

The Problem: Traditional database servers must run 24/7 to wait for users to submit forms or data, which wastes money when the app is idle and crashes if too many users visit at once.

The Solution: A modern backend application that instantly processes data, saves it to a database, scales up automatically for thousands of users, and costs $0 when not in use.

The Tech Stack: Amazon API Gateway acts as the secure front door to accept data from a web form. It forwards the data to an AWS Lambda (Python) function, which processes the information and stores it safely inside an Amazon DynamoDB NoSQL database table.

The Impact: Eliminates the need to manage virtual servers, ensuring 100% uptime and automatic scaling while operating completely within the AWS Free Tier.

Project 3: Cloud-Native Portfolio & Interactive Resume Platform

The Problem: Standard PDF resumes are static and fail to demonstrate hands-on cloud engineering competency, global caching implementation, or strict security configuration.

The Solution: A serverless, globally optimized interactive web application that serves as production-ready proof of core routing, edge-caching, and event-driven data integration.

The Tech Stack: Frontend assets (`index.html`, `style.css`) are hosted as static objects in an Amazon S3 bucket. Amazon CloudFront (CDN) handles global edge caching for low-latency delivery, secured with default SSL/TLS transport-layer certificates. The integrated contact form dynamically invokes an Amazon API Gateway REST pipeline, triggering an AWS Lambda (Node.js backend compute) function to securely handle validation and store user records in an Amazon DynamoDB NoSQL table.

The Impact: Showcases end-to-end cloud infrastructure proficiency. Successfully engineered cross-origin traffic policies by navigating strict browser-level CORS policies and handling complex OPTIONS preflight validations directly at the API Gateway routing layer.
